[문제] 사용자로부터 다음의 데이터를 입력받아 처리한 후 
       재전송하는 프로젝트를 완성하시오. 

날짜(creationDate - date, LocalDate) - 기본값 오늘 날짜 
제품명(productName - text, String) - required 
입출고상태 (status - radio(입고 1, 출고 0), boolean) - 입고 checked
단가 (unitPrice - number, int) - 기본값 0 
개수 (count - number, int) - 기본값 1
세부내용 (details - textarea, String) - 기본값 빈문자열

자바스크립트로 체크할 내용 : 제품명만! 

※ 리소스 사용하기 (image, css, js) - /src/main/resources/static에 위치해 있어야 한다
