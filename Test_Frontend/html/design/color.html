색상(Color)

  • 웹에서의 색상
    • 색상명: 이름을 이용해서 색깔을 지정(ex. blueviolet)
    • RGB: 빨강, 초록, 파랑을 조합해서 사용(10진수, 16진수) - 10진수: 0~255 rgb(0, 255, 125) - 16진수: #00ffef, #00ffee == #0fe
    • RGBA: 빨강, 초록, 파랑 + alpha(불투명도)
      rgba(125,240,98, 0.5);
    • HLS(Hue, Lightness, Saturation): 색상, 명도, 채도를 사용
