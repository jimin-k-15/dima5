CSS Selector(선택자)

HTML 문서의 배경이나 글꼴 등의 서식을 적용하기 위해서는 적용 대상이 되는 문서를 선택해야 한다.
선택자는 문서의 요소를 다양한 방식으로 선택하도록 하는 것이다.
우선순위값이 존재(특이값)
CSS는 head 태그 내에 style 태그를 사용

선택자의 종류

  • 단순 선택자
    • 전체 선택자(*) - 특이값(0)
    • 태그명 선택자 - 특이값(1)
    • 그룹 선택자 - 콤마를 이용해서 여러 개의 요소를 선택해서 동일한 스타일을 적용
    • 클래스 선택자 - class라는 글로벌 속성을 가진 요소를 여러 개 선택 -특이값(10)
    • 아이디 선택자 - id라는 글로벌 속성을 가진 요소를 여러 개 선택 -특이값(100)
    • (예제)단순 선택자 연습
  • 가상(Pseudo) 클래스 선택자
    • 참고 : w3schools
    • 가상 클래스는 태그나 혹은 다른 선택자 뒤에 (콜론)을 붙여 선택을 추가하는 키워드로 선택한 요소가 특별한 상태여야 만족하는 선택자
      주로 a 태그에서 사용, but 다른 태그에서도 사용 가능
    • (예제)가상 클래스 선택자 연습
    • (예제)table 태그에 가상 클래스 선택자 연습
  • 가상 요소(pseudo Element)
    • 참고 : w3schools
    • 가상 요소는 태그의 특정 위치로 지정된 부분에 스타일을 적용할 때 사용
      ::콜론 2개를 이용하여 요소의 일부분에 스타일을 적용 ::first-line, ::first-character, ::before, ::after, ::marker 등이 있음
    • (예제)가상 요소 선택자 연습
  • 속성 선택자 (attribute selector)
    • 태그가 가지는 속성의 타입이나 값을 이용해서 선택
    • 참고 : w3schools
    • [속성]
      [속성=값]
      [속성~=값]
      [속성^=값]
      [속성$=값]
      [속성|=값]
      [속성*=값]
    • (예제)속성 선택자 연습
  • 종속 선택자: 둘 이상의 선택자를 조합해서 사용
      4가지 주요 조합
    • 후손 선택자(공백) : 하위항목 선택
    • 자식 선택자(>) : 바로 밑에 인접한 하위항목 선택
    • 형제 선택자(+) : 상위클래스가 같은 요소이면서 인접
    • 일반형제선택자(~) : 상위클래스가 같은 요소들
    • (예제)종속 선택자 연습
  • 기타 선택자
    • not(선택자): 괄호 안에 선택한 요소를 제외한 것들을 선택
    • nth-child(수식): 수식에 맞는 요소를 선택
    • (예제)기타 선택자 연습
