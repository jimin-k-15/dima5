Position을 이용한 콘텐츠 배치(1)

요소를 특정 위치에 자유롭게 배치할 수 있도록 도와주는 속성
static : (default) - 요소가 본래부터 가지고 있는 배치 형태
absolute : 부모의 (absolute, relative 부모) position 위치를 이용해 배치. Layer를 생성(top, bottom, left, right)
relative : 자신의 position 위치를 이용해 배치. Layer를 생성(top, bottom, left, right)
fixed : 화면을 기준으로 배치. Layer를 생성(top, bottom, left, right)
