Font

  • font-family: 글꼴 지정, 대중글꼴이나 웹폰트를 사용하시오
  • font-size: 글꼴 크기
  • font-weight: 굵기(100~900), 기본값(400)
  • font-variant: 영문자를 기준, 소문자를 기준으로 대문자를 표현

텍스트

  • color: 포그라운드 색상 (글자색)
  • text-decoration: none, underline, overline, through
  • text-transform: 대소문자 변경
  • text-shadow: 그림자
  • white-space: 공백처리
  • direction: 방향
  • text-align: 수평정렬
  • text-justify: 공백처리
  • line-height: 줄간격처리
  • text-overflow: overflow와 함께 사용. 글이 특정 영역의 크기보다 많을 때 처리
