자바스크립트 연산자

※ 실행결과는 콘솔창으로 보시오!

  1. 산술연산자: + - * / % **(거듭제곱 연산자)
  2. 증감연산자: ++, --
  3. 비교연산자: >(초과), <(미만), ≥(이상), ≤(이하),
    == (동등), === (일치), != (유사하지 않음), !== (완전 다름)
  4. 논리연산자: &&, ||, !(단항연산자)
  5. 타입확인 연산자: typeof (오른쪽에 위치한 피연산자의 타입을 확인)
  6. 비트관련 연산자: >>, <<, ~(1's complements) & | ^(XOR)
  7. 대입 연산자: LV = RV (Left Value는 변수만 가능)
  8. 복합대입 연산자: += -= *= ...
  9. 삼항연산자: (조건) ? (참) : (거짓) ==> 참, 거짓의 위치에 문장 허용
  10. 전개 연산자 (spread 연산자): {}나 []에서 사용하며 "...객체"의 형태로 사용
