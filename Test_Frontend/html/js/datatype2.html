  • JSON
    • 데이터를 송수신하기 위한 용도
    • JSON 이전에는 xml을 이용해서 데이터를 송수신했으나, 복잡
    • 구조적인 모습은 {} Object와 동일. 키를 큰따옴표로!

    • JSON.parse()를 이용해서 문자열 -> JSON으로 변환
  • HTML :
    • JS를 이용해서 HTML에 접근하여 구조를 변경할 수 있다.
    • 단, JS가 HTML에 접근하려면 주의해야 한다. DOM구조가 메모리에 구축된 후
    • document.getElementById()와 같은 함수를 이용해서 접근 가능
