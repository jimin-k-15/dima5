하이퍼링크를 표기하는 태그

a 태그를 이용해 홈페이지 내의 문서나 다른 사이트로의 이동이 가능
필수옵션 - href: 이동할 위치의 URL(Uniform Resource Location)
선택옵션 - title: 마우스를 호버시켰을 때 팁이 출력
    target: 링크의 내용이 표시될 창 (_self, _blank, 사용자정의) href의 값 없이 #으로 대체할 경우, 링크 모양만 나타남
닫힌 요소, 인라인 요소, 텍스트가 있으면 텍스트 링크, 이미지가 있으면 이미지 링크
하이퍼링크는 링크상태, active상태, 호버상태, visited상태, focus상태

텍스트링크

  • 네이버
  • 네이트(새 창 열기)
  • 다음(사용자 정의 창 열기)
  • 구글(사용자 정의 창 열기)

이미지링크

특정 위치로 이동

href속성에 #을 이용해서 문서의 특정 위치로 이동 가능
href="#section1" ==> 현재 문서의 아이디가 section으로 지정된 곳으로 이동
href에 넣을 수 있는 값: 다른 URL, 현재 문서의 특정 위치, 다른 문서의 특성 위치
Javascript의 함수 호출도 가능

  • 이동하지 않음
  • 현재 문서의 Section1으로 이동
  • 다른 문서의 Section2로 이동
