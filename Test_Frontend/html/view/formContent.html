폼을 만드는 태그

  • form태그 - action, method(GET, POST), enctype
  • fieldset, legend - 폼의 경계를 만들고 제목을 넣을 때
  • input : type ==> text, password, fild, hidden, submit, reset
  • input : name ==> 서버 쪽 변수명 지정할 때
  • input : type ==> checkbox, radio(같은 그룹의 데이터를 선택), name이 같으면 같은 그룹
  • select : optgroup(선택), option(필수)을 이용해서 드롭다운 방식으로 데이터 선택
  • textarea : 여러줄 글상자, name(필수), cols/rows를 이용해서 크기 조정
  • 기타등등 : 범위의 값, 이메일, 검색창, 색상, 숫자 등의 입력 상자가 있다.
  • global attributes : id(현재 페이지 내에서 유일한 값), class(중복된 값을 부여)
